# Showboat Plugin

Create executable Markdown demos using the [showboat](https://github.com/simonw/showboat) CLI. Capture real command output to produce reproducible proof-of-work documents.

## Commands

- `/demo <feature-name>` — Create a demo document for a specific feature

## Skills

- **showboat-demo** — Showboat CLI workflow patterns and best practices

## Installation

Requires the showboat CLI:

```bash
go install github.com/simonw/showboat@latest
```
